# Earthquake Damage Predictor
https://github.com/EvanGalea2/Earthquake-Damage-Predictor/tree/main
This project uses a variety of Machine Learning models to predict damage to buildings given data about their structure, condition, and location.



Reccommended viewing:

DataExploration: info about dataset and brief analysis
Silvia - Data exploration and vizualization: In depth statistics and analysis
DTree_(260415_V001): Exploration of tree algorithms with feature engineering and PCA
RandomForest_Virginia: Catboost classifier and confusion matrix
RandomForest.ipynb performs grid search on tree based classifiers. It contains the most successful model, as well as submission data


\# Results

Our solution places us 1146th out of 8722 contestants (just outside top 13%) with a score of 0.7405

\#1 Contestant’s solution is better than ours by 2% (0.7558 vs our 0.7405 F1\_score). Median score is 0.7348
